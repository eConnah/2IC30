# 2IC30 Course Repository

This repository contains course materials for 2IC30, focusing on computer architecture concepts including Assembly programming and Register Transfer Language (RTL).

## Directory Structure

- **Assembly/**: Contains assembly language programming exercises, templates, and solutions organized by question sets (Q5, Q6) and programming fundamentals.
- **RTL/**: Includes Register Transfer Language examples and exercise solutions.

## Examples

### RTL Operations
```
ΣA, ΣW, CC ← RB(Bbus), RA(Abus), ALU.cc
RAM [ ΣA ] ← ΣW
RA, CC ← RA(Abus) + RB(Bbus), ALU.cc
```
